# Hospital Management Analysis

## Project Overview
This project analyzes hospital operational data using SQL, Excel, and Power BI to uncover patient trends, billing insights, doctor performance, and revenue KPIs.

## Objectives
- Analyze patient billing patterns
- Track hospital revenue trends
- Evaluate doctor performance
- Monitor treatment statistics
- Build interactive dashboards

## Tools Used
- SQL
- MySQL
- Excel
- Power BI

## Dataset Includes
- Patients
- Billing
- Appointments
- Doctors
- Treatments

## Key Insights
- Top 5 patients generated highest billing revenue
- Cardiology department contributed highest revenue
- Peak appointments occurred during weekdays
- Certain treatments had significantly higher billing trends

## Skills Demonstrated
- Data Cleaning
- SQL Joins
- Aggregations
- KPI Reporting
- Dashboard Design
- Data Visualization
- Business Insights
